This program illustrates how to run a program as a shared library.

The shared library (this file) can be compiled with cat example/03a-shared-library.html | build/html2cpp | g++ -fPIC -shared -x c++ - -o build/03a-shared-library.so

Note that you also need the program that loads the shared library.

The executable can be compiled with cat example/03b-shared-library.cpp | g++ -x c++ -ldl - -o build/03b-shared-library
